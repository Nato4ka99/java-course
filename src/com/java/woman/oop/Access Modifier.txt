Java надає ряд модифікаторів доступу для встановлення рівнів доступу для класів, змінних, методів та конструкторів.

Чотири рівні доступу :
1. Видимий для пакета, за замовчуванням. Не потрібні модифікатори.
2. Видимий лише для класу (private).
3. Видимий для всюдт (public).
4. Видимий для пакету та всіх підкласів (protected).
